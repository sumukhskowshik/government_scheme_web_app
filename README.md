# 🏛 government_scheme_web_app

A role-based Government Schemes Web Application developed using Flask and MySQL.  
The portal allows citizens to explore welfare schemes, check eligibility, apply online, and track applications while admins manage schemes, applications, and user queries.

---

# 🚀 Features

## 👤 Member Features
- Member Registration & Login
- View Government Schemes
- Eligibility-Based Scheme Matching
- Apply for Schemes
- View Application Status
- Add Queries
- View Admin Replies

## 🛠 Admin Features
- Admin Login
- Add Government Schemes
- View All Schemes
- View Applications
- Approve / Reject Applications
- View User Queries
- Reply to Queries

## 🎨 UI Features
- Responsive GOV-style UI
- Dynamic Role-Based Navbar
- Bootstrap 5 Design
- Professional Dashboard Layouts

# ⚙ Installation

## 1. Clone Repository

```bash
git clone https://github.com/sumukhskowshik/government_scheme_web_app.git
```

## 2. Open Project Folder

```bash
cd government_scheme_web_app
```

## 3. Install Requirements

```bash
pip install flask
pip install flask-mysqldb
```

## 4. Configure Database

- Open XAMPP
- Start Apache & MySQL
- Create database:
  government_scheme_db

- Import required tables using phpMyAdmin

## 5. Run Application

```bash
python app.py
```

## 6. Open Browser

```text
http://127.0.0.1:5000
```
